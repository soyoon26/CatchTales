#### Tab 키
![늑대조작키](readme_assets/늑대조작키.jpg)|![빨간모자조작키](readme_assets/빨간모자조작키.jpg)
---|---|

``` Tab을 누르면 조작키를 볼 수 있습니다.```

#### 총파츠 모으기
![총파츠](readme_assets/총파츠.gif)

```빛나고 있는 곳에 가면 총파츠가 있습니다. 빨간 모자가 10개를 모으면 승리합니다.```


#### 빨간모자 얼음
![빨간모자죽음](readme_assets/빨간모자죽음.gif)

```늑대가 빨간모자를 얼리고 30초가 지나기 전에 팀원이 구해지지 않는다면 죽습니다. ```

#### 같은 팀 구하기
![친구구하기](readme_assets/친구구하기.gif)

```얼어있는 팀원에게 다가가 얼음을 깨줍니다.```
#### 빨간모자 승리

<img src="readme_assets/빨간모자승리.gif" width="600" />

``` 총파츠를 10개 다 모으면 빨간 모자가 승리합니다. 다 모으지 못한다면 늑대가 승리합니다. ```
